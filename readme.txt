Getting Started : 

Prerequisites : Node.js , npm

Installation : 

Clone the repository:

git clone https://github.com/gautmsanghvi/Ashley-furniture-clone.git
cd Ashley-furniture-clone


Install dependencies:
npm install

Start the development server:
npm run dev


View in your browser:
Open http://localhost:5173 (or the port specified in your console).

Tech Stack
Vite — build tool & dev server

React — front-end library

TypeScript — static typing

Tailwind CSS — utility-first CSS framework




Folder Overview
src/components: Reusable React UI components.

src/pages: Main website pages.

src/assets: Images and static content.

App.tsx: Main app layout and routing.

main.tsx: Entry point, React app mounting.

Configuration files for Tailwind, Vite, TypeScript, etc.
